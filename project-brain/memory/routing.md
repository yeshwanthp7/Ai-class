# Route Map

- '/': Landing page with Three.js lines and system status.
- '/auth': Combined email/password & Google login portal.
- '/dashboard': Teacher controls, Analytics, Student rosters, and Settings tabs.
- '/session/[code]': Waiting lobby (branched by teacher/student roles).
- '/session/[code]/live': Interactive classroom view with split slide-feed layout.
- '/session/[code]/summary': Post-session analytics scoreboard with CSV export.
