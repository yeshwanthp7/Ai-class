# Class AI Architecture

## Core Tech Stack
- **Framework:** Next.js 16.2.9 (App Router)
- **Language:** TypeScript 5.x
- **Database & Sync:** Firebase Firestore Real-time listeners
- **Video & Voice SFU:** LiveKit Cloud integration
- **AI Core:** Anthropic Claude API (proxied via Groq/Groq API)
- **Eye & Gaze Tracking:** Client-side MediaPipe Face Mesh (with refined iris landmarks)
